## Run

Requires installed python3.x.

* **Windows**

  ```
  python calc.py
  ```
    or run **run.bat**
* **Linux or MacOS**

  ```
  python3 calc.py
  ```
    or run **run.sh**

## Manual

|Symbols| Operation|
|--------|----------|
| + | addition  |
| - |subtraction|
| * |multiplication|
| **  |Exponential|
| / |division|
| // |division, whole part|
|%|mod, rest of division|
|AC|clean a screen|
