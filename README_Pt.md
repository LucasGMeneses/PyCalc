# PyCalc v1.0
Calculadora feita em python com a biblioteca tkinter

## Executar

Requer python3.x instalado.

* **Windows**

  ```
  python pycalc.py
  ```
    ou execute **run.bat**
* **Linux ou MacOS**

  ```
  python3 pycalc.py
  ```
    ou execute **run.sh**
----------------------------
## Manual

|Símbolos| Operações|
|--------|----------|
| + | soma  |
| - |subtração|
| * |multiplicação|
| **  |Exponencial|
| / |divisão|
| // |divisão, parte inteira|
|%|modulo, resto da divisao|
|AC|limpa a tela|
